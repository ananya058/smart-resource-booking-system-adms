# Smart Resource Booking & Conflict-Free Scheduling System  
Using Advanced DBMS Optimization

## 📌 Project Description
This project is a frontend prototype of a Smart Resource Booking System designed to manage resource allocation efficiently while avoiding scheduling conflicts. The system allows users to book resources for specific time slots and prevents overlapping bookings, ensuring optimal utilization of resources.

This version focuses on frontend functionality and basic conflict detection logic as part of ADMS Project Frontend Version 1.

---

## 🛠 Tech Stack
- HTML5  
- CSS3  
- Bootstrap 5  
- JavaScript  

---

## 🚀 Installation & Usage
1. Download or clone this repository.
2. Open `index.html` in any modern web browser.
3. Fill in the booking form to add resource bookings.
4. The system will automatically prevent time conflicts.

---

## 🧠 Features
- Structured booking form
- Conflict-free scheduling validation
- Booking record display in table format
- Clean and responsive UI
- API-ready frontend structure
